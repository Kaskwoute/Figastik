Présentation de Figastik

Site de vente de figurines d'univers fantastiques, réalisé par Antoine Peyrin ASPE.
Utilisation de bootstrap et de ses classes prédéfinis, le javascript n'a seulement été utilisé que pour le responsive du menu.
La font Fontawesome a été utilisé pour ajouter les logos de "inscription","connexion" et "panier".
Utilisation de l'attribut :before :after en css pour l'apparation des barres du menu
